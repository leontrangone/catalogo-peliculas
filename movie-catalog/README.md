# 🎬 Movie Catalog

## Descripción

Aplicación web que permite buscar películas utilizando la API de OMDb.
Los usuarios pueden explorar resultados, ver detalles completos de cada película y guardar sus favoritas.

---

## Funcionalidades

* Búsqueda de películas en tiempo real
* Visualización en tarjetas (cards)
* Página de detalle con información completa
* Sistema de favoritos (guardado en localStorage)
* Modo claro / oscuro persistente
* Animaciones
* Recuperación de última búsqueda

---

## Tecnologías utilizadas

* Astro
* HTML
* CSS
* JavaScript
* OMDb API

---

## Cómo ejecutar el proyecto

1. Instalar dependencias:

```
npm install
```

2. Ejecutar el servidor:

```
npm run dev
```

3. Abrir en el navegador:

```
http://localhost:4321
```

---

## Estructura

* `components/` → componentes reutilizables
* `pages/` → páginas principales
* `styles/` → estilos globales
